##gulpES6ify is *WTF*ify?
###Initial Gulp for develop on ES6 with livereload, babelify, browserify & watchify


###Dependencies
 - gulp
 - gulp-watch
 - gulp-util
 - vinyl-source-stream
 - vinyl-buffer
 - gulp-uglify
 - gulp-sourcemaps
 - browserify
 - watchify
 - babelify
 - gulp-sass
 - eslint
 - connect-livereload
 - gulp-autoprefixer
 - tiny-lr
 - gulp-embedlr
 - gulp-livereload
 - gulp-rename
 - ecstatic
 - ...
 - want more? :cold_sweat:

###Installation
```
npm install gulp -g
npm install
```

###Usage
Edit `gulp/config` with input/output routes
```
gulp
```
open with the browser: `localhost:1337`
and enjoy :satisfied:
