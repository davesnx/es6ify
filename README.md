# es6ify is *WTF*ify?

## Initial Gulp for ES6 with livereload, babelify, browserify & watchify

Just a boilerplate for begin and scale a ECMAScript6 Application.

### Dependencies
 - browserify
 - watchify
 - babelify
 - lodash
 - gulp-livereload
 - gulp-uglify
 - gulp-sourcemaps
 - gulp-ruby-sass
 - gulp-autoprefixer
 - gulp-rename
 - vinyl-source-stream
 - vinyl-buffer
 - node-notifier
 - ...
 - want more? :cold_sweat:

### Installation
```
npm install gulp -g
npm install
```

### Usage
Edit `gulp/config` with input/output routes
```
gulp watch
```
