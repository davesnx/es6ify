###Dependencies
	- babel
	- babelify
	- browser-sync
	- browserify
	- del
	- gulp
	- gulp-autoprefixer
	- gulp-browserify
	- gulp-rename
	- gulp-rsync
	- gulp-sass
	- gulp-sourcemaps
	- gulp-uglify
	- gulp-util
	- node-notifier
	- node-sass
	- vinyl-buffer
	- vinyl-source-stream
	- watchify
 - ...
 - want more? :cold_sweat:

###Installation
```
npm install gulp -g
npm install
```

###Usage

```
gulp
```
